# 3D Heart Design in Onshape

A precise 3D heart model designed from scratch using cloud-based Computer-Aided Design (CAD) on Onshape. This project demonstrates the utilization of foundational sketching tools to construct symmetric, organic 3D shapes.

## 🛠️ Development & Workflow Steps

The model was built systematically by creating a balanced 2D skeleton and transitioning it into a smooth 3D geometry:

### 1. CAD Software & Tools
* **Platform:** Developed on [Onshape](https://www.onshape.com), utilizing its cloud-based parametric design features.
* **Core Functions Used:** Sketching, Symmetric Constraints, and 3D Extrusion/Loft tools.

### 2. Implementation Sequence
* **Step 1: Reference Lines:** Established the vertical and horizontal construction lines to maintain perfect symmetry between both halves of the heart.
* **Step 2: Curving with 3-Point Arc:** Utilized the **3-Point Curve** tool to carefully draw the smooth, organic curvature of the upper lobes and the outer boundaries of the heart.
* **Step 3: Sketch Closure & Constraints:** Fully defined the sketch by connecting the curve endpoints to the center axis to ensure a watertight profile.
* **Step 4: 3D Transformation:** Transformed the completed 2D sketch into a solid 3D heart model using the dimensional tools in Onshape.

## 🎯 Project Conclusion
The project successfully highlights the transition from fundamental 2D sketching—relying heavily on precise curves and construction lines—to complex organic 3D shapes. Designing this model serves as an excellent practice for mastering parametric constraints and surface geometries in CAD.
